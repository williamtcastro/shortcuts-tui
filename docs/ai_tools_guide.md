# 🤖 AI & Agent Tools Guide

## 🧠 Claude Code
| Alias | Action |
| --- | --- |
| `cc` | Start Claude session |
| `ccc` | Continue last conversation |
| `ccr` | Resume from history |
| `ask <text>` | One-shot prompt |
| `aireview` | AI code review of current branch |
| `explain <cmd>` | Explain a shell command |

## ♊ Gemini CLI
| Alias | Action |
| --- | --- |
| `gmc` | Start Gemini session |
| `gmcc` | Continue last conversation |
| `gmccr` | Resume from history |
| `gmask <text>` | One-shot prompt |
| `gmreview` | AI code review vs main |

## 🛠 Agents & Editors
| Alias | Program | Action |
