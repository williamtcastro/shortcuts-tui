# 👻 Ghostty Shortcuts (Fast Modern Terminal)
*Ghostty comes with built-in vim-like splitting and seamless integration with Tmux & Neovim.*

## ⌨️ General Commands
| Keybind | Action |
| --- | --- |
| `Alt + s` | Toggle **Quick Terminal** (Drop-down) |
| `Cmd + c / v` | Copy / Paste from clipboard |
| `Cmd + q` | Quit Ghostty |
| `Cmd + w` | Close Surface/Tab |
| `Cmd + t` | New Tab |

## 🪟 Splits & Panes
| Keybind | Action |
| --- | --- |
| `Cmd + d` | Create a new split **Right** |
| `Cmd + Shift + d` | Create a new split **Down** |

## 🧭 Navigation (Panes & Prompts)
| Keybind | Action |
| --- | --- |
| `Cmd + Shift + Up` | Jump to **previous** prompt |
| `Cmd + Shift + Down` | Jump to **next** prompt |
| `Cmd + h/j/k/l` | Seamless navigation between Ghostty, Tmux & Nvim! |
