# 🟢 Tmux Shortcuts & Commands
*Prefix key is configured to `Ctrl + a` (C-a)*

## 🪟 Windows & Panes
| Keybind | Action |
| --- | --- |
| `C-a c` | Create a new window |
| `Alt + H` | Go to previous window (no prefix) |
| `Alt + L` | Go to next window (no prefix) |
| `C-a "` | Split window horizontally (Top/Bottom) |
| `C-a %` | Split window vertically (Left/Right) |
| `C-a x` | Kill current pane |
| `C-a z` | Zoom current pane to full screen |

## 🧭 Navigation (Vim-Tmux-Navigator)
*Use these to seamlessly move between Tmux panes and Neovim splits*
| Keybind | Action |
| --- | --- |
| `Ctrl + h` | Move Left |
| `Ctrl + j` | Move Down |
| `Ctrl + k` | Move Up |
| `Ctrl + l` | Move Right |

## 📋 Copy Mode & Scrolling
| Keybind | Action |
| --- | --- |
| `C-a [` | Enter Copy Mode (Vi keys) |
| `v` | Begin Selection (in copy mode) |
| `C-v` | Toggle Rectangle Selection (in copy mode) |
| `y` | Copy Selection & Cancel (in copy mode) |
